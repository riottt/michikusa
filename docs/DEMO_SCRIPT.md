# 審査デモ台本

## 90秒版

### 0–10秒: 問題

「休日なのに行きたい場所が思いつかず、検索する気力もない。外出中に予定が終わった後も同じです。MICHIKUSAは、その空白時間をAIへ渡します。」

画面は淡い地図と「今日を動かす」だけを見せます。

### 10–25秒: Agent稼働

「ボタンを一度押すと、Google ADKの18ノードが動きます。」

- Calendar空き時間
- Places候補
- 移動範囲
- 過去履歴

が地図上で並行して現れ、候補が絞られ、2〜4本のピンが落ちます。

### 25–40秒: 一日の決定

「AIは候補を並べません。行き先、順番、各地点で何をするか、帰る時刻まで決めます。」

完成カードで時間、予算、帰宅目安、地点数を示します。

### 40–52秒: Calendar Action

「この道草で出発」を押します。

「移動、書店で読む時間、休憩、帰宅がGoogleカレンダーへ一括登録されます。書き込みは人が承認した後だけです。」

### 52–67秒: 現地の遊び

最初のピンへ移動し、「着いた」を押します。

「場所だけでなく、表紙だけで一冊選ぶ、10分読む、一曲聴くといった小さな遊びまで作ります。」

LUCK獲得を見せます。

### 67–80秒: 再計画

「15分遅れた」を選びます。

「7ノードの再計画グラフが、完了した地点を残し、次の予定に間に合うよう残りの道とCalendarを更新します。」

### 80–90秒: 記憶と締め

共有カードを表示します。

「MICHIKUSAは、おすすめ検索ではありません。行きたい場所がない日に、現実の一日を動かす外出エージェントです。」

> 行き先は考えなくていい。行くかどうかだけ、決めて。
